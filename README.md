# Day Planner Mobile App – UI/UX Design

link - https://www.figma.com/proto/ACTDGYmwYiW8s4yQpZWhC6/Untitled?node-id=0-1&t=pkeYpSmvEL9vgOim-1

Designed a modern and user-friendly Day Planner mobile application focused on helping users organize their daily activities, schedules, and personal journals in one place. The design emphasizes simplicity, productivity, and an intuitive user experience.

Key Features -
User Login & Registration Screens
Interactive Calendar View
Daily Task Management
Schedule Planning with Date & Time Selection
Progress Tracking Dashboard
Personal Journal Section
Task Status Monitoring (Completed, In Progress)
Clean and Consistent Mobile UI Design

Design Highlights -
Created a complete mobile app workflow from onboarding to task management.
Focused on accessibility, visual hierarchy, and ease of navigation.
Used a consistent color palette and modern card-based layouts.
Designed responsive screens suitable for Android devices.
Applied basic UX principles to improve usability and user engagement.

Tools Used -

Figma
UI Design
Wireframing
Prototyping
User Experience (UX) Design
